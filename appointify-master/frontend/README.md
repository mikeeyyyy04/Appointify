# Appointify Frontend

Modern web frontend and mobile app for the Appointify clinic management system.

## Tech Stack

- **Framework**: SvelteKit
- **Styling**: Tailwind CSS
- **UI Components**: Bits UI, Lucide Icons
- **Mobile**: Capacitor (Android)
- **Language**: TypeScript

## Installation

```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

Visit `http://localhost:5173` (or the port shown in terminal)

## Building

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Mobile Development

For Android development with network configuration sync:
```bash
npm run dev:cap
```

Build and sync with Capacitor:
```bash
npm run build:cap
```

Open in Android Studio:
```bash
npx cap open android
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run check` - Type check with Svelte
- `npm run lint` - Lint code
- `npm run format` - Format code with Prettier
- `npm run dev:cap` - Dev server with Capacitor network sync
- `npm run build:cap` - Build and sync with Capacitor

## Project Structure

- `src/routes/` - SvelteKit routes (pages)
- `src/lib/components/` - Reusable Svelte components
- `src/lib/requests/` - API request functions
- `src/lib/stores/` - Svelte stores for state management
- `src/lib/types/` - TypeScript type definitions
- `android/` - Android native project (generated by Capacitor)

## Environment Variables

If you need to configure the API URL, create a `.env` file:

```env
PUBLIC_API_URL=http://localhost:7325
```

## Backend Connection

By default, the frontend connects to `http://localhost:7325`. Make sure the backend is running before starting the frontend.

See the main [README.md](../README.md) for complete setup instructions.
